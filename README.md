# PersonalDashboard - In Progress

A full‑stack personal dashboard that blends a clean FastAPI backend with a visually rich React frontend. It presents key metrics, recent activity, goals, and skills in a single polished view.

## Overview
This project showcases a lightweight dashboard experience with a dedicated API layer, real‑time‑ready data fetching, and a modern UI. It’s designed to be easy to run locally and easy to extend.

## Features
- FastAPI backend with structured endpoints for dashboard data
- React frontend with bold visual styling and a focused layout
- Responsive design optimized for desktop and mobile
- Automated API and frontend utility tests
- Git automation scripts for consistent sync and history hygiene

## Tech Stack
- Backend: Python, FastAPI, Uvicorn
- Frontend: React, Vite
- Testing: Pytest, Vitest
